# dna_visualizer
The DNA Sequence Analysis Web Application is an interactive bioinformatics tool built using 'Python' and 'Streamlit'.  
It allows users to perform key DNA sequence analyses such as:

- Generating reverse complements
- Calculating GC content
- Performing AT/GC skew analysis

This project helps students, researchers, and bioinformatics enthusiasts explore the structure, composition, and asymmetry of DNA sequences in a simple web interface.
